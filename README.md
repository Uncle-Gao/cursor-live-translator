# Cursor-Live-Translator：AI 驱动的实时本地化引擎

> [!NOTE]
> **原项目 `cursor-settings-zh` 已正式更名为 `Cursor-Live-Translator`。**
> 此次更名标志着项目从单纯的设置汉化脚本，进化为支持 AI 实时刷新的全场景本地化引擎。

本项目是专门为 Cursor 定制的高性能实时翻译引擎。通过 **V2.3 Trusted Bootstrap Injection（受信任引导挂载）** 架构，实现了离线字典匹配、动态变量正则解析与多引擎 AI 实时翻译的深度融合，为您提供全方位的中文开发体验。

> [!IMPORTANT]
> **全系架构跨代升级：运行时 DOM 拦截与 AI 异步实时刷新**
> 这是 V2 PRO 架构的革命性突破。为解决传统汉化插件“逢更新必黑屏”且“无法翻译 AI 动态内容”的缺陷，我们采用了外挂系统模式，直接**内联追加至受信任的 `workbench.js` 引导程序**。随后利用 `MutationObserver` 在内存渲染层实时汉化 UI。在最新的 **V2.3 版本** 中，我们引入了 AI 结果异步回传刷新机制，彻底消除了首屏翻译的视觉延迟感。

## 🌟 核心亮点 (Pro Features)

### 1. AI 异步实时刷新 (Direct UI Refresh) 🚀 `[V2.3 重磅更新]`
解决了以往 AI 翻译需要二次扫描才能生效的痛点：
- **无感更新**：当 AI 翻译结果从后台返回时，引擎会自动微调局部 DOM，让原本显示的英文瞬间“跳变”为优雅的中文。
- **视觉连贯**：无需刷新窗口，无需切表，翻译结果即回传即生效。

### 2. 精位屏蔽系统 (Refined Shielding) 🛡️
为了绝对保障代码原生体验，V2.3 预设了极为精准的屏蔽链：
- **代码区隔离**：`.view-lines` 确保您的每一行代码都保持原生英文，不受汉化引擎任何干扰。
- **路径保护**：`.monaco-breadcrumbs` 自动跳过编辑器顶部的路径导航，防止文件路径被误翻。
- **列表优化**：`.monaco-list-row` 精准屏蔽侧边栏列表，确保资源管理器与搜索结果的纯净。

### 3. 动态变量正则匹配 (M4 Smart Pattern)
内置正则模式预编译引擎：
- **智能捕获**：自动识别 `Found (\d+) files` 等带变量的动态长句。
- **精准映射**：将捕获组实时填入汉化模板（如：`找到 $1 个文件`）。

### 4. 全平台多引擎支持 (Multi-Engine Support)
- **主流适配**：原生支持 **DeepSeek (推荐)**, **OpenAI (GPT-4o/mini)**, **通义千问 (Qwen)**, **Kimi (Moonshot)**, **DeepL API**。
- **点对点通信**：翻译请求由本地直接发往官方 API，无任何中转，确保您的 API Key 与隐私数据绝对安全。

## 🚀 快速开始

### 1. 环境准备
确保已安装 Node.js 环境。

### 2. 运行安装程序
在终端执行：
```bash
node cursor_zh_pro.js
```

### 3. 操作指引
1. 选择 `4. 管理汉化屏蔽规则` -> `R. 恢复默认配置`（确保应用最新的 V2.3 精准规则）。
2. 选择 `3. 配置 AI 翻译引擎` 并录入您的 API Key。
3. 选择 `1. 一键汉化` 并重启 Cursor 即可体验。

## 🛡️ 安全性与隐私说明

1. **API Key 本地化**：存储在本地家目录 `~/.cursor_live_translator/` 之中，不随代码上传。
2. **直连通信**：不设立中转服务器，请求直达官方接口。
3. **安全隔离**：默认跳过所有代码区 (`<pre>`, `<code>`, `.view-lines`) 及输入框，**绝不触碰您的源码**。

## 🛠️ 常见问题 (Troubleshooting)

### 1. 汉化后 Cursor 提示“损坏”？
这是 macOS 的签名保护。请运行：
```bash
sudo xattr -cr /Applications/Cursor.app
sudo codesign --force --deep --sign - /Applications/Cursor.app
```

### 2. 如何确认引擎已启动？
按下 `Cmd + Option + I` (Mac) 或 `Ctrl + Shift + I` (Win) 查看控制台。若看到紫色印记 `[Cursor-Live-Translator] 多引擎动力系统就绪`，则代表 V2.3 已完美生效！
